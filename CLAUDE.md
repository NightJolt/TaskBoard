# TaskBoard - Task Management Platform

Admin invites users. Users authenticate, create projects, create tasks, and assign them to project members.

### Task Fields
Title, Status (Todo / In Progress / Done), Deadline, Priority

## Structure
- `backend/` — NestJS modular monolith (see `backend/CLAUDE.md`)
- `frontend/` — Angular 19 + Angular Material (see `frontend/CLAUDE.md`)

## Git Conventions
- One logical change per commit
- Descriptive messages explaining "why"
